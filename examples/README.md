# Dossier Examples

Ce dossier contient des exemples et des résultats visuels du modèle PokeBrAIn.

## Contenu

### Images de test
Ce dossier est destiné à contenir des images de cartes Pokémon utilisées pour tester le modèle de classification.

### Résultats visuels du modèle
Les résultats de l'entraînement et de l'évaluation du modèle seront stockés ici, notamment :
- **Matrices de confusion** : visualisation des prédictions du modèle par rapport aux classes réelles
- **Courbes de pertes** (loss curves) : évolution de la fonction de perte durant l'entraînement
- **Courbes de précision** : évolution de la précision (accuracy) durant l'entraînement
- Autres métriques et visualisations pertinentes

## Structure suggérée
```
examples/
├── test_images/          # Images de test
├── confusion_matrices/   # Matrices de confusion
├── loss_curves/          # Courbes de pertes
└── other_metrics/        # Autres visualisations
```
