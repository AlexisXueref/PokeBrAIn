# Dossier Examples

Ce dossier regroupe des exemples visuels illustrant le fonctionnement du projet PokéBrAIn : il permet de montrer concrètement les cas de test et les résultats obtenus avec l'IA développée.

## Contenu actuel

- **Example_card_data_augmented.png** : Exemple d'image de carte Pokémon après data augmentation (luminosité, rotation, etc.) utilisée lors de l'entraînement du modèle.
- **Example_tests_cards.pdf** : Exemples de vraies cartes Pokémon utilisées pour valider le modèle à la fin (tests sur des cas concrets).
- **Loss Evolution.png** : Courbe montrant l'évolution de la perte (`loss`) pendant l'entraînement du modèle (permet d'analyser la convergence et la performance).
- **sample_confusion_matrix.png** : Exemple de matrice de confusion illustrant la capacité du modèle à classer correctement les cartes (analyse des erreurs et réussites).

***

## Utilisation

- **Images de test** : Servez-vous de ces exemples pour comprendre comment le modèle a été validé sur des cartes réelles.
- **Cartes data-augmentées** : Permet d'illustrer le prétraitement des données d'entraînement via augmentation artificielle.
- **Figures analytiques (loss, confusion matrix, etc.)** : Donnent une visibilité sur les performances et puissances du modèle.
